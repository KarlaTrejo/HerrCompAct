Modificaciones al juego paint.py

Se agregaron las siguientes funcionalidades

- circle: Funcion que permite que se forme un circulo en el rango(tamaño) y posición dado por 2 puntos que hace el usuario
- Rentangle: Funcion que permite formar un rentangulo en la posicion y el rango(tamaño) dados de dos puntos ingresados por el usuario
- Triangle: Funcion que permite formar un rectangulo un triangulo equilatero en la posición y rango (posicion) dados por el usuario
- Yellow: color amarillo agregado a la lista de colores disponibles

Ninguna de las funcionalidades que estaban antes se combiaron o modificaron.
